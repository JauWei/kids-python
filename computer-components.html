CPU 中央處理器

像電腦的大腦，負責執行指令、做計算，並協調其他零件一起工作。

零件控制台

CPU 深入認識

Central Processing Unit

    一個按鍵如何變成畫面

    1. 輸入鍵盤、滑鼠或觸控板送出訊號。
    2. 暫存RAM 放著程式正在需要的資料。
    3. 計算CPU 判斷下一步要做什麼。
    4. 圖像GPU 準備要顯示的畫面。
    5. 輸出螢幕、喇叭或印表機呈現結果。

    AI 不是突然變聰明，而是靠資料、模型、訓練方法和大量運算一步一步學出規律。它看到新的問題時，會用已學到的模式推測最可能的答案。
